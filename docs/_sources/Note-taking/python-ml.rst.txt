============================
基于Python的机器学习系统设计
============================

第三章 将数据变为信息
-----------------------

数据探索的对象是数据的 **成分** 和数据的 **结构** 。

并行化
    将学习算法分解为自适应查询序列，并且并行地处理这些查询；或在多台机器上运行同一算法，每台机器采用不同的参数配置。Map reduce的开源版本就是Hadoop。

流式处理
    过量、过快的输入数据难以存储，就必要在数据流层面进行分析。

数据模型
    描述数据是如何组织的。

    - Structure （行和列；节点和边；键值对）
    - Constraints （“个数”只能为自然数值）
    - Operations （根据指定键值寻找特定值）

数据来源
    数据库、HTML文档、自然语言、图像、API、信号（如声音）。

数据清洗
    - 处理缺失数据；
    - 导入数据（如从pdf文件中）；
    - 数据类型转换。

数据可视化
    - 明确数据类型后将其映射为视觉元素：位置、大小、纹理、颜色、方向、形状；
    - 考虑视觉元素的感知特性（最敏感到最不敏感）：位置、长度、角度、坡度、面积、体积、颜色、密度。
